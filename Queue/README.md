# 队列
和栈相反,队列是一种**先进先出**(first input first out 缩写为 FIFO)的线性表,它只允许在表的一端进行插入,而在另一端删除元素.在队列中,允许插入的一端叫做**队尾***,允许删除的一端则称为**队头**

## 队列的基本操作

- 构造一个空队列
- 销毁队列
- 清空队列
- 判断队列是否为空
- 获取队列的长度
- 获取队列的队头元素
- 插入元素到队尾
- 删除队头元素
- 循环遍历元素
