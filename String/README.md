# 串
计算机中的非数值处理的对象基本上是字符串数据.随着语言加工程序的发展,产生了字符串处理.这样字符串也就作为一种变量类型出现在越来越多的程序设计语言中,
同时也产生了一系列字符串的操作.字符串一般简称为串.在汇编和语言的编译过程中,源程序及目标程序都是字符串数据.

## 串的定义
串(或字符串)是由零个或多个字符组成的有限序列.串中字符的数目 n 称为串的长度.零个字符的串称为空串,它的长度为 0
