## 数组
数组是数据结构中最基础的数据结构，几乎每一种编程语言都有数组。

### 什么是数组？
所谓数组，是一组有序元素的序列，是用来存储多个相同类型数据的集合。数组会在内存中开辟一段连续的空间，并在此空间存放元素。通俗的来讲，数组就像公寓里的一排房间，每个房间都有一个编号（0,1,2,.....100），根据房间的编号，可以找到房间里的人。

![array](../images/array.png)

数组属于线性结构。
